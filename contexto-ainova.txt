ESTADO DEL PROYECTO - AInova POS
========================================
Última actualización: 19/06/2026 ~16:30

=== DESCRIPCIÓN ===
Sistema POS multi-negocio. Cada persona tiene su propio
usuario/contraseña (como red social). Cada usuario es dueño
de un negocio con datos separados (productos, inventario,
empleados, historial).

=== TECNOLOGÍA ===
- Frontend: React 18 + Vite 5
- Iconos: Lucide React
- Persistencia: localStorage (prefijo pos_{businessId}_)
- Sin backend ni base de datos externa
